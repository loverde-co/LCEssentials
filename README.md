## Installation

#### CocoaPods
You can use [CocoaPods](http://cocoapods.org/) to install `LCEssentials` by adding it to your `Podfile`:

```ruby
platform :ios, '9.0'
use_frameworks!
pod 'LCEssentials'
```

To get the full benefits import `YourLibrary` wherever you import UIKit

``` swift
import LCEssentials
```
